# Date
Date 对象基于 1970年1月1日（世界标准时间）起的毫秒数。

## 构造函数
```js
new Date();
new Date(value);
new Date(dateString);
new Date(year, month[, day[, hour[, minutes[, seconds[, milliseconds]]]]]);
```
注意：只能通过调用 Date 构造函数来实例化日期对象，如果不加 new ，则会返回一个字符串，而不是日期对象。
Date对象没有字面值格式。
