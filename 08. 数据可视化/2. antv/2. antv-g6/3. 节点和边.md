# data
Graph中的data 只有两种类型，node 和 edge。
对于单个 item 上没有 setItemState, show, hide 方法。
原因在于用户在很多场景下需要监听item的状态，G6 上的事件监听仅在 graph 上开放。
对于 node 和 edge 的更新操作，只能使用 graph 上的增删改查方法。

# node 和 edge 通用方法

* getContainer()：返回元素的图形容器

* set(name, value)
