# splash window
在app完成加载显示一个，向导页面。  
```js
let splashWin = new BrowserWindow({
  ...
  frame: false,
  resizable: false,
  alwaysOnTop: true,
  backgroundColor: '#FFF',
  show: false,
})
splashWin.once('ready-to-show', () => {
  splashWin.show();
})

```
