# splash window 策略
在app完成数据准备和加载之前显示一个向导页面过渡，这个页面称为 splash window。    
```js
let splashWin = new BrowserWindow({
  ...
  frame: false,
  resizable: false,
  alwaysOnTop: true,
  backgroundColor: '#FFF',
  show: false,
})
splashWin.once('ready-to-show', () => {
  splashWin.show();
})

```
